Renewable Electricity World Map - 2014

Filter :

First Filter : Selects Country which uses 'selected' Renewable Resources to produce electricity

Second Filter: Changes the map based on value 'selected'.
'Carbon Emission' Will show Co2 Emitted by each country in Carat(kt)
'Total Electricity Production' Will show production of Electricity(with and without renewable sources) by each country

Click: A pie chart for each country will appear, showing all the renewable resources used to produce electricity

NEXT PAGE: TIME LINE
The Map shows, % Electricity produced by renewable resorces(excluding HydroPower) used each country

Timeline: This percentage can be seen from 1990 to 2014

Top 5: Top 5 countries with high percentage can be seen in the bar chart

Filter : Countries based on their income group can be selected, with all the above functionality
